Arbeitsverzeichnis wechseln

{{< highlight bash>}}cd {{< / highlight>}}

Mit cd (change directory) kann man das Arbeitsverzeichnis wechseln. Der Pfad kann absolut (wie in > cd C:/Users/User/Documents) oder relativ (wie in > cd Documents) sein.

Besondere Bedutung haben die Pfade . und ...
. zeigt auf das aktuelle Verzeichnis, > cd . macht also nichts. .. zeigt auf das übergeordnete Verzeichnis, > cd .. wechselt also in das übergeordnete Verzeichnis.

Verzeichnis auflisten

{{< highlight bash>}}ls [-a] [-l] [pfad]{{< / highlight>}} {{< highlight powershell>}}dir{{< / highlight>}}

Den Inhalt eines Verzeichnisses auflisten. Die Option -a zeigt auch versteckte Dateien an, -l die Listenansicht. Windows zeigt die Listenansicht standardmäßig an, Linux und Mac die Kurzansicht. zeigt mehr Informationen an. Ohne Pfad wird das aktuelle Verzeichnis aufgelistet.

Verzeichnis erstellen

{{< highlight bash>}}mkdir [-p] {{< / highlight>}}

Mit mkdir (make directory) kann man ein neues Verzeichnis erstellen. Der Pfad kann absolut oder relativ sein.

Neue Datei erstellen

{{< highlight bash>}}touch {{< / highlight>}} {{< highlight powershell>}}ni {{< / highlight>}}

Erstellt eine leere Datei unter dem angegebenen Pfad.

Ist die Dati vohranden, aktualisiert touch das Änderungsdatum auf die aktuelle Zeit.

Datei verschieben

{{< highlight bash>}}mv {{< / highlight>}}

Einen Ordner oder eine Datei verschieben. <source> ist der Quellpfad, <dest> der Zielort.

Datei löschen

{{< highlight bash>}}rm [-r] {{< / highlight>}}

Das Verzeichnis oder die Datei <node> löschen. Die Option -r löscht rekursiv, also auch Verzeichnisse und deren Inhalt, unter Windows muss man zum Löschen von Verzeichnissen stattdessen {{< highlight powershell>}}rmdir {{< / highlight>}} verwenden. Vorsicht! Gelöschte Dateien können nicht einfach wiederhergestellt werden.

Inhalt der Datei ausgeben

{{< highlight bash>}}cat {{< / highlight>}}

Inhalt der Datei <datei> ausgeben.

Programme starten

Auch grafische Programme können über die Kommandozeile gestartet werden. Der Befehl ist dabei normalerweise der Name des Programms. {{< highlight bash>}}code {{< / highlight>}}

Startet Visual Studio Code und öffnet die Datei oder das Verzeichnis <node>. Jedoch ist die Syntax je nach Programm unterschiedlich, manche Programme benötigen zusätzliche Parameter, andere andere Befehle: {{< highlight bash>}}start notepad++ {{< / highlight>}}

Exit

{{< highlight bash>}}exit{{< / highlight>}}

Das Terminal schließen.
