Arbeitsverzeichnis

Das Arbeitsverzeichnis ist das Verzeichnis, in dem sich der Benutzer gerade befindet. Alle relativen Pfade werden von diesem Verzeichnis aus interpretiert. Jedes Programm, dass im Terminal ausgeführt wird, bekommt automatisch das Arbeitsverzeichnis als Input-Parameter übermittelt.

Normalerweise steht das Arbeitsverzeichnis zu Anfang der Kommandozeile, z.B. C:\Users\User oder user@mylinux:~$, aber mit dem Befehl pwd (print working directory) kann man sich den absoluten Pfad zum aktuellen Verzeichnis auch anzeigen lassen.

Skript

Ein Shell-Skript ist eine Datei, die eine Abfolge von Terminal-Befehlen enthält. Diese Befehle werden nacheinander ausgeführt, wenn das Skript ausgeführt wird. Shell-Skripte werden häufig verwendet, um wiederkehrende Aufgaben zu automatisieren. In Windows können die Skripte die Endungen .bat, .cmd oder .ps1 bzw. .ps2 haben. Unter Linux wird normalerweise die Endung .sh verwendet.

Vorsicht! Weil Shell-Kommandos direkt auf dem System ausgeführt werden, können Skripte auch Schaden anrichten. Verwenden Sie nur Skripte, die Sie verstehen und denen Sie vertrauen.

Absolute und relative Pfade

Ein Pfad ist eine Zeichenfolge, die den Speicherort einer Datei oder eines Verzeichnisses auf einem Dateisystem angibt. Es gibt zwei Arten von Pfaden: absolute und relative Pfade.

Ein absoluter Pfad beginnt immer am Stammverzeichnis des Dateisystems und enthält den vollständigen Pfad zur Datei oder zum Verzeichnis (in Windows inklusive des Laufwerksbuchstabens), so wie etwa C:\Users\User\.gitconfig oder /etc/passwd.

Ein relativer Pfad beginnt im aktuellen Arbeitsverzeichnis und enthält nur den Pfad zur Datei oder zum Verzeichnis relativ zum aktuellen Arbeitsverzeichnis, z.B. Documents\file.txt oder ../Pictures/photo.jpg.

Alle Befehle, die einen Pfad benötigen, können i.d.R. sowohl absolute als auch relative Pfade verwenden.

TAB-Taste

Damit man im Teminal nicht alles eintippen muss, vervollständigt die TAB-Taste automatisch Befehle und vor allem Datei- und Verzeichnispfade, wenn ein Teil des Pfades eingegeben wurde. Beispiel: Steht im aktuellen Terminal folgendes: {{< highlight powershell>}}> C:\Users\User\>cd Doc{{< /highlight>}}

vervollsständigt die TAB-Taste den Befehl automatisch zu {{< highlight powershell>}}> C:\Users\User\>cd Documents{{< /highlight>}}
