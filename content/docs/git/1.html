Git ist ein Tool zum Versionieren von und kollaborativen Arbeiten an Textdateien. Dabei ist von zentraler Bedeutung, dass

  • Dateistände zu bestimmten Zeitpunkten festgehalten und wiederhergestellt werden können,
  • Mehrere Personen an einem Projekt arbeiten können, ohne sich gegenseitig zu behindern.

Git wird in der Regel über die Kommandozeile bedient. Es gibt jedoch auch grafische Oberflächen, die die Bedienung vereinfachen.

Hilfe zu Git gibt es unter git-scm.com, in der Kommandozeile mit dem Befehl git help <kommando>. Die Einführung folgt grob dem Vorgehen von git help everyday ( online ).
